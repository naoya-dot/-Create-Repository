# 図面 寸法チェッカー v3

スマートフォンでPDF・JPEG・PNG・DXF図面を開き、既知寸法で縮尺を合わせて2点間を測定するPWAです。

## 対応形式
- PDF（複数ページ）
- JPEG / JPG
- PNG
- DXF（ASCII形式の2D図面）

DXFで表示できる主要図形は LINE、LWPOLYLINE / POLYLINE、CIRCLE、ARC です。複雑なブロック、文字、ハッチ、3D図形などは省略される場合があります。

## 操作
1. 図面を選ぶ
2. 二本指で拡大・縮小・移動
3. 「基準寸法を設定」から2点を選び、実寸mmを入力
4. 「寸法を測る」から測定する2点を選ぶ
5. 十字をドラッグし、ルーペを見ながら微調整
